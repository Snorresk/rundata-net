About Rundata-net

Welcome to Rundata.info, a web client for the Scandinavian Runic-text Data Base (SRDB). The database contains information about Scandinavian runic inscriptions. The website is called Rundata.info, while the project itself is named Rundata-net. Although the interface is in English, much of the database content and many descriptions remain in Swedish.

Rundata.info offers two responsive layouts. The desktop layout provides the complete interface and is designed for larger screens. The mobile and tablet layout is adapted for touchscreens and smaller displays, with simplified search tools and easier access to the map and inscription information.

An AI assistant for searching the database is currently under development. Its results may be incomplete or inaccurate and should therefore be verified against the inscription records.

Rundata-net was created by Vadim Frolov and Sofia Pereswetoff-Morath. Rundata.info is available free of charge.

Rundata-net Code and User Support

The source code of Rundata-net is freely available under the GNU GPL in the Rundata-net GitHub repository. You can run Rundata-net on your own computer, on a dedicated server, or in the cloud.

To report a problem, please use the project issue tracker. For other comments, contact us at mail@vadimfrolov.com or spm@su.se. User documentation is available on the Help page.

Browser Support

Rundata.info is designed for modern versions of Microsoft Edge, Chrome, Firefox, and Safari. Safari 16.6 is supported. Internet Explorer is not supported.
